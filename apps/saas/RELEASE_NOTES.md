# Release Notes

## v0.1.0 - Public Launch Baseline

Date: 2026-04-09

### Highlights

- Product-facing public landing and signup flow.
- Customer portal for usage and credential self-management.
- Privacy subject endpoints (data access/export/deletion).
- Onboarding endpoints with site provisioning flow.
- Enterprise auth options (magic link, SSO bridge, OIDC).
- Billing integration with Stripe checkout/webhooks and alert escalation.
- Deployment resources for Render, including environment validation script.

### Notes

- This release establishes the public launch baseline for demos and first paid pilots.
- For production rollout, ensure strict production env settings and webhook/domain configuration.
